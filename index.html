SISTEM INFORMASI BANK MINI IQTI BERBASIS ANDROID (Bank Sekolah)

Membangun sistem informasi nasabah berbasis android yang dapat menampilkan saldo, notifikasi transaksi, info akun dan transfer antar nasabah.

Sistem informasi admin berbasis web dengan menggunakan aplikasi codeigniter sebagai framework PHP,  Jquery dan MySQL sebagai database. serta framework IONIC untuk android dengan dasar pemrograman fungsi pendaftaran nasabah, transaksi kegiatan menabung, pembuatan laporan.

Sistem informasi PPDB 2020

Software ini telah digunakan di SMK Maarif NU Tirto dan berjalan lancar. pembuat aplikasi ini ialah R. Anggit Herdianto anda bisa mendownload nya di: http://github.com/AnggitHerdianto si pemilik tidak menghendaki jual beli aplikasi tersebut. aplikasi ini melingkupi :
1. Pendaftaran siswa baru
2. Pengecekan data siswa baru
3. Pengaturan jadwal pendaftaran
4. Pengaturan admin dan superadmin

  • Mengganti tabel pada database nama panggilan menjadi NIK
  • Menyesuaikan halaman export siswa pendaftaran dengan database
  • Menambahkan menu cetak formulir pendaftaran
